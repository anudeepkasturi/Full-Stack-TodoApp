# API Endpoints

## HTML
### Root
- `GET /  (loads React App)`

## JSON

### Users
- `POST /api/users`
- `PATCH /api/users`

### Session
- `POST /api/session`
- `DELETE /api/session`
- `GET /api/session`

### List
- `GET /api/lists (search)`
- `POST /api/lists`
- `GET /api/lists/:id`
- `PATCH /api/lists/:id`
- `DELETE /api/lists/:id`

### Tasks
- `GET /api/tasks`
- `POST /api/tasks`
- `GET /api/task/:id`
- `DELETE /api/task/:id`
